## Overview

This directory contains benchmark results for the Policy Engine canonical benchmark suite.

### Running Benchmarks

To generate benchmark results, run:

```bash
cd policy-engine-benchmarks
cargo run --release --bin run_benchmarks
```

### Output Structure

- `summary.md` - This file, containing human-readable benchmark summary
- `raw/` - Directory containing raw JSON benchmark data
  - `all_results.json` - Combined results from all benchmark targets
  - `<target_id>.json` - Individual result files per benchmark target

### Benchmark Targets

| Target ID | Description |
|-----------|-------------|
| `policy_evaluation` | Core policy evaluation against request contexts |
| `condition_parsing` | Parsing and evaluation of policy conditions |
| `decision_tree_resolution` | Traversal and resolution of policy decision trees |
| `policy_bundle_loading` | Loading and parsing of policy bundles |
